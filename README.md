# TagTool-SublimeText
A port of https://github.com/EnashMods/TagTool-NPP for Sublime Text IDE.
This is syntax highlighting, block grouping, and auto completion for TagTool command scripts for Sublime Text IDE.

How to Install
----------

Copy ``TagTool.sublime-completions`` and ``TagTool.sublime-syntax`` files into your pc ``%appdata%\Sublime Text\Packages\User`` folder.
Alternatively you can just paste the path into your address bar and it should take you to the correct location.
